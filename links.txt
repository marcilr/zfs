ZFS on linux
Very useful page
The native Linux kernel port of the ZFS filesystem.  To get started with ZFS on Linux simply
download the latest release and install using the directions for your distribution.
Debian jessie 8.1 has packages:
  libguestfs-zfs - guest disk image management system - ZFS support
  parted - disk partition manipulator
  simplesnap - Simple and powerful network transmission of ZFS snapshots
  zfs-fuse - ZFS on FUSE
  zfsnap - Automatic snapshot creation and removal for ZFS
http://zfsonlinux.org/
